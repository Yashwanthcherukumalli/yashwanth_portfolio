Next Hire

An online job portal where employers can post jobs and job seekers can apply. The platform includes modules for Employers, Candidates, and Consultancies, allowing seamless interaction between different user roles.

Built using Angular 14 with TypeScript, HTML5, CSS3, and Bootstrap, the platform integrates REST APIs for real-time data updates and efficient state management.

Key features include job search with advanced filters, profile creation and updates, real-time notifications, and secure login using JWT-based authentication.

Dependency injection and lazy loading techniques were applied to enhance performance. The UI was designed with reusable components following Agile development methodology.

  • Built using Angular 14, HTML5, CSS3, Bootstrap
  • Implemented lazy loading and dependency injection
  • Integrated REST APIs for real-time updates
